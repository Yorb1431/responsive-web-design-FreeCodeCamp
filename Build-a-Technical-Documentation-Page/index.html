Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It is a small and lightweight language that enables interactive web pages.

Core features include:

  • First-class functions
  • Prototype-based object orientation
  • Dynamic typing
function greet(name) { return "Hello " + name; }
What you should already know

This guide assumes you have basic programming knowledge.

Familiarity with web technologies is recommended:

  • HTML fundamentals
  • CSS basics

Prior experience with any programming language is helpful but not required.

const x = 42;
JavaScript and Java

Despite similar names, JavaScript and Java are fundamentally different.

Key differences include:

  • Execution environment
  • Type systems
  • Object models

JavaScript is primarily used for client-side scripting, while Java is often used for server-side applications.

// JavaScript (dynamic typing) let x = 10; x = "text"; // Valid // Java (static typing) int x = 10; x = "text"; // Error
Hello world

To create your first JavaScript program:

console.log("Hello World!");

This simple statement outputs text to the console.

You can run this code in:

  • Web browser developer tools
  • Node.js environment
  • Online code editors
Variables

Variables are containers for storing data values.

JavaScript has three declaration types:

  • var (function scope)
  • let (block scope)
  • const (block scope, immutable)

Modern JavaScript prefers let and const over var.

Variable names must follow these rules:

let userName = "John"; // Valid let 123age = 25; // Invalid
